# Hi, I'm Phobos

Also known as `Phobos(1641)?` around most of the web, or **Lysander Sokolov** if you prefer something a little more professional.

## About Me

I'm a hobbyist programmer who's been messing around with **C++** for about 12 years now--though
whether that actually makes me *good* at it is very much up for debate.
**Python** is my other main language, and I occasionally dabble in web development,
though frontend isn't really my passion.

My coding strategy is to aggressively throw code at the compiler until it works. (Only half-joking.)

## Tools I Use

- **Licensing:** AGPL for software, CC BY-NC-SA for everything else.
